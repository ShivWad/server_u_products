# U_products

server_u_products is the backend part for my U_products project

## Major tools which I used
* **NodeJs**
* **ExpressJs**
* **MongoDB** && **Mongoose**
* **Firebase** (to store images)
* **Bcryptjs**


## Included Services
* **User**
* **Product**

## Installation

Use yarn to install dependencies

```bash
yarn
```

## Usage
Run the server using yarn command
```bash
yarn dev
```
